# 🏦 AI-Powered Loan Decision Support System

An end-to-end **Machine Learning** application that predicts loan approval using an **AdaBoost Classifier**. The project is deployed as an interactive **Streamlit** web application, enabling users to analyze model details, explore dataset insights, visualize feature importance, and receive real-time loan approval predictions with downloadable reports.

## 📌 Project Overview

Financial institutions receive thousands of loan applications every day. Evaluating each application manually can be time-consuming and inconsistent. This project leverages Machine Learning to automate the loan approval process by analyzing applicant information and predicting whether a loan should be approved.

The application not only predicts loan approval but also provides model insights, dataset statistics, feature importance visualization, and downloadable prediction reports through an interactive web interface.

# Installation

### Clone the Repository

```bash
git clone https://github.com/Vivek9942/AI-Powered-Loan-Decision-Support-System.git
```

### Navigate to Project Directory

```bash
cd AI-Powered-Loan-Decision-Support-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
